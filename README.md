# L1Cache
This repository broadly will cover all the verilog files involved in designing a L1 Cache


For estimation of area, latency, and power of my cache design, I used the cacti tool from HP : https://github.com/HewlettPackard/cacti

Here's how my project journey started:

    1) I started by refreshing myself with my Semester II course : Computer Architecture
    2) I followed Prof. Smruti R. Sarangi's course on Operating Systems and Computer Architecture to get a better understanding of this rather complicated domain
    3) I started reading through Chapter 11 of his textbook, https://www.cse.iitd.ac.in/~srsarangi/archbook/chapters/memory-system.pdf
    4) Using (3) I could write my own L1 cache as a verilog module
